# Linear Issue Flow

Complete Linear issue workflow.

## Available Actions

### Start Work
- Get my assigned issues from Linear
- Select highest priority issue
- Create feature branch
- Update issue status to "In Progress"

### During Development
- Check issue requirements
- Add progress comments
- Link commits to issue

### Ready for Review
- Update status to "In Review"
- Create PR with issue link
- Add preview URL from Vercel

### Complete Issue
- Update status to "Done"
- Add deployment URL
- Check for related issues
